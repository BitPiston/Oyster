Warning

This library is considered pre-alpha.

This library provides access to Paypal's NVP api.

Public Functions

All functions take either a single argument or a hash of arguments (NOT a hashref).

All functions return hashrefs containing the nvp response.

new

pdt

Sends a Payment Data Transfer request to Paypal

For an explanation of the variables returned by this, see PayPal's variable reference.

This uses a different authentication scheme than the NVP-based API functions.

PDT also returns data in a different format than NVP, so it must use a custom parser.

Prototype:

                hashref = obj->pdt(string transaction_id)

get_transaction_details

transaction_search

parsed_transaction_search

A wrapper for transaction_search() that parses the results and puts them into a more meaningful data structure.

validate_ipn

Private Functions

_send_msg

This is the main work horse of this library, this sends an NVP message and returns the response.

Prototype:

                 $obj->_send_msg(string method, hashref params)
